Yes – these days, we don’t have enough time measuring devices like mobiles, microwaves, computers – I wanted to build another one. It was my first Arduino project so the task wasn’t as easy as I expected. In the theory, we need only pass time & data from RTC to 7-Segment Display. Simple isn’t it? I found plenty of tutorials, had so many small issues like – digits wasn’t displayed correctly (mixed cables).
